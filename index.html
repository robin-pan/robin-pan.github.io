Calculator


Originally, this project was a command line arithmetic calculator which I made using C++ as an OOP exercise. I then decided to add GUI, but could not find an appropriate C++ library. Thus, I decided to learn C# and made both the front-end and back-end.


This calculator takes an arithmetic expression in infix format, converts the input to postfix format using the Shunting-yard algorithm, and evaluates the expression. Each mathematical operation is set up polymorphically, allowing for the easy addition of new mathematical operators.
